# Sentin4e Usage Guide

## Basic Usage

Run Sentin4e by invoking the command line interface:

```bash
sentin4e
```

## Global Options

- `--debug`: Enable verbose debug logging, which includes full tracebacks on errors.
- `-v`, `--version`: Show the dynamic version dashboard.
- `-h`, `--help`: Show the dynamic help system.

### Dealing with Errors

If Sentin4e crashes unexpectedly, you can run it with the `--debug` flag to see the full Python traceback and identify the root cause:

```bash
sentin4e <command> --debug
```
